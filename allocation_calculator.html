📖 Why These Allocations

1
Core is your wealth engine (default 70%). Long-term compounding does the heavy lifting. Options income is supplementary — never let it reduce your Core exposure.
2
Wheel bucket (default 15%) generates CC income. You need 100 shares minimum per CC contract. Build Wheel shares via put assignment first, then start selling calls.
3
BPS collateral (default 15%) earns income without owning shares. Runs in parallel with Core. All premium reinvests into Core — accelerating compounding.
4
55/45 SPY/QQQ split. SPY provides stability anchor. QQQ overweighted for growth. Adjust with the slider based on your risk tolerance.
5
All options income reinvests into Core. A $600/month BPS income reinvested into SPY/QQQ Core compounds for 20 years. That’s the real wealth builder — not the $600 itself.

Building Phase — When Wheel Shares Are Not Yet Enough

Phase 1
Run BPS only. Use sell put strategy to accumulate Wheel shares via assignment. No CC yet — not enough shares per contract.
Phase 2
100+ Wheel shares per ticker. Start selling 1 CC contract per bucket. BPS continues in parallel.
Phase 3
200+ Wheel shares. Full strategy running. All premium income flows into Core. Rebalance quarterly.
Reinvestment loop: BPS premium earned → buy more SPY/QQQ Core shares CC premium earned → buy more SPY/QQQ Core shares Wheel assignment → cash back to BPS/sell put cycle → get assigned again at lower price → those shares join Core or Wheel Every dollar of options income compounding in Core at 10% average annual return × 20 years = 6.7× growth. The premium is not the wealth — the compounded shares are.
